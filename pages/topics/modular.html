Modular Arithmetic
Last Updated: 07/28/2022

What is the modulo operator?

When you divide two integers A and B, you get an equation that looks like this:
$$\frac{A}{B}=Q\ remainder\ R$$
Where:
A is the dividend
B is the divisor
Q is the quotient
R is the remainder
$$A\ mod\ B = R$$

Sometimes, we are only interested in R, or the remainder, instead of the quotient. For these cases, we use the modulo operator, or mod. While modular arithmetic is a cornerstone of number theory and discrete mathematics, it has applications beyond math, such as in computer science, with hashing functions and cryptography. By the end of this article, you will learn how to quickly evaluate expressions such as $12^{134}\operatorname{mod}7$. Have fun!

Example: $$\frac{11}{3}=3\ remainder\ 2$$ Therefore: $$11\ \operatorname{mod}\ 3\ =\ 2$$

Modular Arithmetic with Negative Numbers

Negative numbers are something that you cannot avoid in math in general, so this section will show you how to perform the modulo operation when there is a negative dividend. The remainder returned by the modulo operator must always be a positive number, so when there is a negative dividend, we have to "overshoot" and multiply our divisor by a number that will result in a product lower than our dividend. Then we have to see how far away we landed from our dividend, and that difference is the remainder.

Another method to calculate the modulo with a negative dividend is to repeatedly add the divisor to the dividend until we get a number that is between 0 and (the divisor - 1).
$$-11\operatorname{mod}7$$

Method 1:

$$7\cdot\left(-2\right)=-14$$ $$-11-\left(-14\right)=3$$ $$-11\operatorname{mod}7=3$$

Method 2:

$$-11+7\left(2\right)=3$$ $$-11\operatorname{mod}7=3$$

Modular Addition and Subtraction

$\left(A+B\right)\operatorname{mod}C=\left(A\operatorname{mod}C+B\operatorname{mod}C\right)\operatorname{mod}C\\$
Example: A = 14, B = 17, C = 3

$\left(14+17\right)\operatorname{mod}3=\left(14\operatorname{mod}3+17\operatorname{mod}3\right)\operatorname{mod}3\\$
$\left(14+17\right)\operatorname{mod}3=\left(2+2\right)\operatorname{mod}3\\$
$\left(14+17\right)\operatorname{mod}3=4\operatorname{mod}3=1\enspace\checkmark\\$
$31\operatorname{mod}3=1\enspace\checkmark\\$
This property is also observed with modular subtraction:

$\left(A-B\right)\operatorname{mod}C=\left(A\operatorname{mod}C-B\operatorname{mod}C\right)\operatorname{mod}C\\$
Example: A = 37, B = 18, C = 5

$\left(37-18\right)\operatorname{mod}5=\left(37\operatorname{mod}5-18\operatorname{mod}5\right)\operatorname{mod}5\\$
$\left(37-18\right)\operatorname{mod}5=\left(2-3\right)\operatorname{mod}5\\$
$\left(37-18\right)\operatorname{mod}5=-1\operatorname{mod}5=4\enspace\checkmark\\$
$19\operatorname{mod}5=4\enspace\checkmark\\$

Modular Multiplication

$\left(A\cdot B\right)\operatorname{mod}C=\left(A\operatorname{mod}C\cdot B\operatorname{mod}C\right)\operatorname{mod}C\\$
Example: A = 19, B = 17, C = 8

$\left(19\cdot17\right)\operatorname{mod}8=\left(19\operatorname{mod}8\cdot17\operatorname{mod}8\right)\operatorname{mod}8\\$
$\left(19\cdot17\right)\operatorname{mod}8=\left(3\cdot1\right)\operatorname{mod}8\\$
$\left(19\cdot17\right)\operatorname{mod}8=3\operatorname{mod}8=3\enspace\checkmark\\$
$323\operatorname{mod}8=3\enspace\checkmark\\$
This property will be extremely helpful later on in modular exponentiation!

Modular Exponentiation

$A^{B}\operatorname{mod}C=\left(\left(A\operatorname{mod}C\right)^{B}\right)\operatorname{mod}C\\$
Example: A = 9, B = 5, C = 7

$9^{5}\operatorname{mod}7=\left(\left(9\operatorname{mod}7\right)^{5}\right)\operatorname{mod}7\\$
$9^{5}\operatorname{mod}7=\left(2^{5}\right)\operatorname{mod}7\\$
$9^{5}\operatorname{mod}7=32\operatorname{mod}7=4\enspace\checkmark\\$
$9^{5}\operatorname{mod}7=4\enspace\checkmark\\$
However, $\left(A\operatorname{mod}C\right)^{B}$ can become very large for even relatively low values of B, too large for even calculators to calculate, so by itself, the property's use is very limited. We can overcome this shortcoming by combining this property with the Modular Multiplication property, which we previously learned, by splitting up large exponents into smaller exponents using the Multiplication Property of Exponents:

$a^{\left(x+y\right)}=a^{x}\cdot a^{y}\\$
Example:
$2^{8}=2^{5}\cdot2^{3}\\$
Example using the Modular Multiplication property with this:

$7^{4}\operatorname{mod}5\\$
can be broken up into:

$\left(7^{2}\cdot7^{2}\right)\operatorname{mod}5\\$
and solved as:

$\left(\left(7^{2}\operatorname{mod}5\right)\cdot\left(7^{2}\operatorname{mod}5\right)\right)\operatorname{mod}5\\$
$\left(4\cdot4\right)\operatorname{mod}5=1\\$
Now, going back to what we were trying to simplify, $\left(A\operatorname{mod}C\right)^{B}$. We can break up B into a sum of numbers that are powers of 2 by converting it to binary (our base conversion topic page), as it's easy to find $A^{B}\operatorname{mod}C$ when B is a power of 2, as you will see in the example, where we work through the problem that we promised that you would learn how to solve at the beginning of the article:

$12^{134}\operatorname{mod}7$

First, we have to break 134 into powers of 2, or convert it into binary:

$134_{10}=10000110_{2}\\$
$134=128+4+2\\$
$12^{134}=12^{128}\cdot12^{4}\cdot12^{2}\\$
Now, we have to find $12^{2}\operatorname{mod}7$, $12^{4}\operatorname{mod}7$, and $12^{128}\operatorname{mod}7$

$12\operatorname{mod}7=5\\$
$12^{2}\operatorname{mod}7=\left(12\operatorname{mod}7\right)^{2}\operatorname{mod}7=25\operatorname{mod}7=4$
$12^{4}\operatorname{mod}7=\left(12^{2}\operatorname{mod}7\right)^{2}\operatorname{mod}7=16\operatorname{mod}7=2$
$12^{8}\operatorname{mod}7=\left(12^{4}\operatorname{mod}7\right)^{2}\operatorname{mod}7=4\operatorname{mod}7=4$
$12^{16}\operatorname{mod}7=\left(12^{8}\operatorname{mod}7\right)^{2}\operatorname{mod}7=16\operatorname{mod}7=2$
$12^{32}\operatorname{mod}7=\left(12^{16}\operatorname{mod}7\right)^{2}\operatorname{mod}7=4\operatorname{mod}7=4$
$12^{64}\operatorname{mod}7=\left(12^{32}\operatorname{mod}7\right)^{2}\operatorname{mod}7=16\operatorname{mod}7=2$
$12^{128}\operatorname{mod}7=\left(12^{64}\operatorname{mod}7\right)^{2}\operatorname{mod}7=4\operatorname{mod}7=4$
(Note: A pattern like the 4, 2, 4, 2... pattern that you might have noticed in this example won't show up in every problem like this.)

Now, we put it all back together with the modular multiplication property:

$12^{134}\operatorname{mod}7=\left(12^{128}\cdot12^{4}\cdot12^{2}\right)\operatorname{mod}7\\$
$=\left(\left(12^{128}\operatorname{mod}7\right)\cdot\left(12^{4}\operatorname{mod}7\right)\cdot\left(12^{2}\operatorname{mod}7\right)\right)\operatorname{mod}7$

Now, we substitute in the values that we found:

$=\left(4\cdot2\cdot4\right)\operatorname{mod}7\\$
$=32\operatorname{mod}7\\$
$=4\enspace\checkmark\\$
